¿A quiénes está dirigida esta especialización?

Esta especialización está diseñada para profesionales con título universitario en disciplinas como:

  • Ingeniería de software
  • Ingeniería de sistemas
  • Ingeniería informática
  • Ingeniería electrónica
  • Ingeniería eléctrica
  • Ingeniería mecatrónica

Oportunidades profesionales para graduados

Prepárate para acceder a diversas oportunidades profesionales en empresas y organizaciones de todos los sectores, a nivel local, regional y nacional. Algunos de los roles destacados incluyen:

  • Líder/Ingeniero de calidad /Pruebas/QA
  • Analista de pruebas/Funcional
  • Tester de software
  • QA Automation/Analyst/Engineer
  • Consultor de calidad
